# VetDiary

Мобильное приложение для владельцев питомцев. Позволяет вести историю болезни, отслеживать график прививок и находить ближайшие ветеринарные клиники на интерактивной карте.
Проект доработан в рамках Практической работы №5 (Безопасность и публикация).

## Технологии
* Язык программирования: Java
* База данных: SQLite (ORM-библиотека Room)
* Картография: OSMDroid (OpenStreetMap)
* Сетевые запросы: HttpURLConnection, Open-Meteo API
* Пользовательский интерфейс: XML

## План монетизации и публикации

### Монетизация
1. Freemium-модель: Базовый функционал (локальный дневник, карта, погода) доступен пользователям бесплатно. Подписка "VetDiary Premium" будет открывать облачную синхронизацию данных между устройствами, онлайн-консультации с ветеринарами и расширенную аналитику здоровья питомца.
2. Партнерские программы: Размещение на карте партнерских ветеринарных клиник и зоомагазинов с приоритетным выделением их маркеров (рекламная монетизация).

### Публикация
1. Целевые платформы: RuStore (основная площадка для целевой аудитории РФ) и Google Play.
2. Этапы релиза:
   - Закрытое альфа-тестирование (проверка стабильности базы данных и работы карты).
   - Открытое бета-тестирование.
   - Полноценный релиз версии 1.0.
3. Подготовка материалов: Разработка графических ассетов (иконка приложения, скриншоты интерфейса), составление Политики конфиденциальности (Privacy Policy), ASO-оптимизация (ключевые слова: "дневник питомца", "ветеринар", "прививки").

## Обеспечение безопасности приложения
В текущей версии реализованы следующие меры безопасности:
1. Изоляция данных (Sandbox): Все пользовательские записи (история болезни, прививки) хранятся в локальной базе данных SQLite. На уровне ОС Android эта база данных находится во внутренней, защищенной директории приложения, к которой не имеют доступа другие программы.
2. Сетевая безопасность: Обмен данными с внешним сервисом (Open-Meteo) осуществляется исключительно по защищенному протоколу HTTPS, что предотвращает перехват трафика (Man-in-the-Middle атаки).
3. Принцип наименьших привилегий: Приложение запрашивает в AndroidManifest только минимально необходимый набор разрешений (INTERNET и ACCESS_NETWORK_STATE), не требуя доступа к конфиденциальным данным пользователя (камера, контакты, микрофон, точная GPS-геолокация устройства).

## Инструкция по сборке APK из командной строки
Для сборки установочного .apk файла средствами интерфейса командной строки выполните следующие шаги:

1. Откройте корневую папку проекта VetDiary в терминале.
2. Убедитесь, что у вас установлен JDK и прописаны переменные среды (JAVA_HOME).
3. Выполните команду сборки (в зависимости от вашей ОС):
   * Windows (PowerShell): .\gradlew assembleDebug
   * Windows (CMD): gradlew assembleDebug
   * macOS / Linux: ./gradlew assembleDebug
4. Дождитесь появления сообщения BUILD SUCCESSFUL.
5. Готовый установочный файл app-debug.apk будет находиться по пути:
   app/build/outputs/apk/debug/app-debug.apk
