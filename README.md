# VetDiary

Мобильное приложение для владельцев питомцев. Позволяет вести историю болезни, отслеживать график прививок и находить ближайшие ветеринарные клиники на карте.

Проект разработан в рамках Практической работы №2 (Реализация пользовательского интерфейса).

## Технологии
* **Язык:** Java
* **UI:** XML (ConstraintLayout, LinearLayout)
* **Архитектура:** Статический интерфейс (без подключения реальной БД на данном этапе)
* **Среда разработки:** Android Studio
* **Дизайн:** [Макет в Figma]

## Функционал (UI)
1. Главный экран с виджетом погоды.
2. Экран истории болезни (карточки записей).
3. Экран графика прививок.
4. Карта ветеринарных клиник.
5. Формы добавления новых записей (история/прививки).
6. Навигация между экранами через нижнее меню (Bottom Navigation) и систему вкладок.

## Инструкция по сборке APK из командной строки
Для сборки установочного `.apk` файла средствами интерфейса командной строки выполните следующие шаги:

1. Склонируйте репозиторий и откройте корневую папку проекта `VetDiary` в терминале.
2. Убедитесь, что у вас установлен JDK и прописаны переменные среды.
3. Выполните команду сборки (в зависимости от вашей ОС):
    * **Windows (PowerShell):** `.\gradlew assembleDebug`
    * **Windows (CMD):** `gradlew assembleDebug`
    * **macOS / Linux:** `./gradlew assembleDebug`
4. Дождитесь сообщения `BUILD SUCCESSFUL`.
5. Готовый файл `app-debug.apk` будет находиться по пути:
   `app/build/outputs/apk/debug/app-debug.apk`
