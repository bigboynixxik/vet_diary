# VetDiary

Мобильное приложение для владельцев питомцев. Позволяет вести историю болезни, отслеживать график прививок и находить ближайшие ветеринарные клиники на карте.

Проект разработан в рамках Практической работы №3 (Использование локальной базы данных).

## Технологии
* Язык программирования: Java
* База данных: SQLite (ORM-библиотека Room)
* Пользовательский интерфейс: XML (ConstraintLayout, LinearLayout)
* Среда разработки: Android Studio

## Функционал приложения
1. Локальная база данных (Room) для хранения записей.
2. Полный набор CRUD-операций (создание, чтение, обновление, удаление) для модуля "История болезни".
3. Полный набор CRUD-операций для модуля "Прививки".
4. Главный экран и карта ветеринарных клиник (статический UI).
5. Навигация между экранами через нижнее меню (Bottom Navigation) и систему вкладок.

## Инструкция по сборке APK из командной строки
Для сборки установочного .apk файла средствами интерфейса командной строки выполните следующие шаги:

1. Откройте корневую папку проекта VetDiary в терминале.
2. Убедитесь, что у вас установлен JDK и прописаны переменные среды (JAVA_HOME).
3. Выполните команду сборки (в зависимости от вашей операционной системы):
   * Windows (PowerShell): .\gradlew assembleDebug
   * Windows (CMD): gradlew assembleDebug
   * macOS / Linux: ./gradlew assembleDebug
4. Дождитесь появления сообщения BUILD SUCCESSFUL.
5. Готовый установочный файл app-debug.apk будет находиться в директории проекта по пути:
   app/build/outputs/apk/debug/app-debug.apk
